# Login & Dashboard Demo

Simple login UI with a protected dashboard that shows login history stored in `localStorage`.

## Features

- Modern glassmorphism-style UI
- Email + password validation
- Password show/hide toggle
- Stores successful logins (email, password, time) in browser `localStorage`
- Dashboard unlock protected by a dashboard password (hardcoded in `script.js`)

## Running

Just open `index.html` in a browser.

## Deploying to GitHub Pages

1. Push this project to a GitHub repository.
2. Go to **Settings ➝ Pages**.
3. Under **Source**, choose `Deploy from a branch`.
4. Select branch (e.g. `main`) and folder `/ (root)`.
5. Save – GitHub will give you a URL where the site is live.
